# 💥 Install Instructions

## 💻 Neovim

 > Install requires Neovim Nightly. Always review the code before installing a configuration.

Clone the repository and install the plugins:

```sh
git clone git@github.com:kiet231199/neovim ~/.config/kiet231199/neovim
```

## 🔥 Set neovim to be your git editor
Add these codes to your .bashrc and .zshrc (depend on your shell)
```bash
if [ -n "$NVIM_LISTEN_ADDRESS" ]; then
    alias nvim=nvr -cc split --remote-wait +'set bufhidden=wipe'
fi

if [ -n "$NVIM_LISTEN_ADDRESS" ]; then
    export VISUAL="nvr -cc split --remote-wait +'set bufhidden=wipe'"
    export EDITOR="nvr -cc split --remote-wait +'set bufhidden=wipe'"
else
    export VISUAL="nvim"
    export EDITOR="nvim"
fi
```
